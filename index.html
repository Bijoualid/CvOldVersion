Compétences

Dans le cadre de mon parcours scorlaire, voici mes différentes compétences :

Wordpress

Wordpress est un système de gestion de contenu libre écrit en PHP et reposant sur une base de données MySQL. WordPress est utilisé comme moteur de blog, mais ses fonctionnalités lui permettent également de gérer n'importe quel site web. Wordpress est aussi le système de publication le plus utilisé au monde. Actuellement, 29% des sites internet s'appuient sur cette technologie, et ce chiffre ne cesse d'augmenter.

Gestion de projet

Un projet est un ensemble d'activités coordonnées et maîtrisées pour atteindre un objectif en respectant des exigences de qualité, de coûts et de délais. Pour parvenir à gérer un projet, j'ai appris à réaliser la rédaction de : notes de cadrage, propositions commerciales, et cahiers des charges.

HTML CSS

Le language html est la fondation de tous les sites internet, il est utilisé pour structurer une page web. Le language CSS quant à lui, spécifie la mise en page, les couleurs et les polices de caractères. Apprendre ces deux languages, constitue donc le point de départ de mon apprentissage.

Bootstrap

Bootstrap est librarie CSS la plus utilisée, développée par les développeur de Twitter, Bootstrap est un code déjà écrit, et réutilisable dans nos projets web.

PHP

PHP est un langage de script côté serveur qui est utilisé pour développer des sites Web statiques ou dynamiques ou des applications Web.

Github

GitHub est un service en ligne qui permet d'héberger ses repositories de code. GitHub est un outil gratuit pour héberger du code open source, et propose également des plans payants pour les projets de code privés. C'est donc un outil vraiment utile qui est essentiel au flux de travail d'un développeur.

Javascript

JavaScript est un langage de programmation pour le web. Il est principalement utilisé pour améliorer les pages Web afin d'offrir une expérience plus conviviale. Son apprentissage anisi que celui de sa bibliothèque Jquery font donc partie des languages incontournables du web.

Symfony

Symfony est un ensemble de composants PHP réutilisables et un framework PHP pour les projets web.
